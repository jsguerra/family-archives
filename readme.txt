=== Family Archives ===
Tags: genealogy, family tree, family history
Requires at least: 3.9
Tested up to: 4.7.3

Family Archives is a plugin which allows for the creation of basic genealogy profiles. You can create page profiles for family members and ancestors with facts, bio summaries and more.

== Description ==

With Family Archives plugin you will get an ancestry page auto-generated. This will list every profile you create. Individual profiles can be accessed via 'domain/ancestry/profile_slug/'.

== Installation ==

1. Install and Activate as you would any other plugin
2. A new post type called Ancestry will appear under 'Posts'. You can add profiles there.

== Frequently Asked Questions ==

= Why do I need to type basic HTML code into the editor? =

The plugin is still being developed. At this stage this will allow you to format your data for each profile page.

= How do I display a Pedigree on my profile page? =

The pedigree is based on 'emperor.js' a Javascript script for embedding pedigree charts into blog posts. To add a tree simply use the hash marks followed by a name:
# first name
## parent name
### grand parent name
### grand parent name
## parent name
### grand parent name
### grand parent name

You can learn more about Emperor here: (https://github.com/bencrowder/emperor)
